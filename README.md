# Collins Munene — Personal Portfolio Website

A fully responsive, modern portfolio website built with Bootstrap 4, featuring a dark cyber/AI aesthetic theme.

1. **Services** — AI-Powered Cybersecurity, Full-Stack Development, Machine Learning & Data Science
2. **Projects** — RobustNIDS, AegisFlow AI, NeuralBastion
3. **Academics** — BSc Computer Science journey at University of Debrecen (2023–2026)
4. **Favourites** — Basketball, Chess, Horse Riding, Music, Food & Art
5. **Contact** — Contact form with validation

##  Tech Stack

- HTML5, CSS3, JavaScript
- Bootstrap 4
- jQuery with smooth scrolling
- Font Awesome 5 icons
- Google Fonts (Sora, Space Grotesk, JetBrains Mono)
- Canvas-based particle animation
- Scroll-triggered reveal animations
